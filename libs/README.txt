================================================================================
                    TacZ Integration - Setup Instructions
================================================================================

Для включения интеграции с модом TacZ (возможность сбивать Шахеды оружием):

1. Скачайте мод TacZ для Minecraft 1.20.1 Forge
   - Официальная страница: https://www.curseforge.com/minecraft/mc-mods/timeless-and-classics-zero

2. Скопируйте JAR файл TacZ в эту папку (libs/)
   - Файл должен называться: tacz-1.20.1-xxx.jar (где xxx - версия)
   - Пример: tacz-1.20.1-1.0.3-all.jar

3. Пересоберите мод командой: gradlew build

================================================================================
                         Configuration (ata-common.toml)
================================================================================

После установки, настройте параметры в конфиге:

[Anti-Air Defense (TacZ Integration)]
# Включить возможность сбивать Шахеды
shahedShootdownEnabled = true

# Максимальное здоровье Шахеда (100 = по умолчанию)
shahedMaxHealth = 100.0

# Глобальный множитель урона (1.0 = нормальный)
shahedDamageMultiplier = 1.0

# Множитель урона при попадании в двигатель (задняя часть)
shahedEngineDamageMultiplier = 1.5

# Множитель урона при попадании в боеголовку (передняя часть)
# Может вызвать преждевременную детонацию!
shahedWarheadDamageMultiplier = 2.0

# Минимальный порог урона (фильтрует слабое оружие)
shahedMinCaliberDamage = 2.0

# Показывать дым при повреждении
shahedSmokeOnDamage = true

# Попадание в боеголовку при низком HP может вызвать детонацию
shahedCriticalDetonation = true

================================================================================
                              Hit Zones
================================================================================

Шахед имеет три зоны попадания:

1. WARHEAD (Боеголовка) - передняя часть
   - Множитель урона: x2.0 (настраивается)
   - При HP < 30% есть 30% шанс преждевременной детонации

2. ENGINE (Двигатель) - задняя часть  
   - Множитель урона: x1.5 (настраивается)
   - Вызывает дополнительный дым

3. BODY (Корпус) - средняя часть
   - Множитель урона: x1.0
   - Стандартный урон

================================================================================
                            Damage Effects
================================================================================

- При HP < 100%: Лёгкий дым из двигателя
- При HP < 50%:  Густой чёрный дым и пламя
- При HP = 0:    Уничтожение (взрыв в воздухе без урона по земле)
